Services

Professor Class Schedule

The system contain the information about the daily scheduled lectures of that professor. This information will include the course name and the timing of the lectures and contain the links of the lecture slides for each week for a particular batch.

Student Records

The system include the points awarded to the student for each lab submission, viva and lab attendance. The professor can also retrieve the overall performance status of a particular student for the final evaluation and can draw an evaluation graph for a student.

TA Lab Schedule

The system contain the information about the lab scheduled for TAs. The information will include the course name and the timing of the lectures and contain the links of the lecture slides for each week for a particular batch.

About

  • Evaluation

    “Professor’s Support System” will provide assistance to the Professor in handling the details of each and every student of a batch for the fair evaluation at the end of the semester. This project will help the professor in maintaining the record of the performances given by any particular student.

  • Class Schedule

    This system will also contain the information about the daily scheduled lectures of that professor when they logged in with their id. This information will include the course name and the timing of the lectures.

  • Leave Form

    This project also includes a online leave form for the professors, TAs and the employes who are getting salaries, So that whenever they need a leave they can directly send the leave form to the senior authority and take permission for the holiday.

  • Attendance Record

    keeping a record of attendance is really useful. The project will contain the information about the total number of classes held in a week and the number of classes attended by a particular student. This will help in retrieving the percentage attendance of any student till date.
